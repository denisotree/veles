# اختصارات لوحة المفاتيح وأوامر الشرطة المائلة في TUI

> 🌐 **اللغات:** [English](../../en/reference/tui.md) · [Русский](../../ru/reference/tui.md)

يفتح `veles tui` (أو `veles` المجرّد) واجهة REPL التفاعلية. وهي محادثة قابلة
للتمرير للخلف مع مُحرِّر متعدّد الأسطر، وشريط حالة، ومفتّش قابل للطي.

## اختصارات لوحة المفاتيح

| المفتاح | الإجراء |
|---|---|
| `Ctrl+D` | الخروج |
| `Ctrl+C` | نسخ آخر رد للمساعد؛ اضغط مرتين خلال 1.5 ثانية للخروج |
| `Ctrl+V` | اللصق من الحافظة |
| `Ctrl+Shift+C` / `⌘C` | نسخ التحديد الحالي (OSC52). على Terminal.app في macOS، يعمل التحديد الأصلي بالسحب + ⌘C مباشرةً |
| `Ctrl+I` | تبديل المفتّش (الاستدلال، نشاط الأدوات، سجل الرموز/الأخطاء) |
| `Ctrl+R` | فتح مُنتقي الجلسات (استئناف جلسة سابقة) |
| `Ctrl+T` | فتح مُنتقي السمات |
| `Shift+Tab` | تدوير وضع التشغيل: `auto → planning → writing → goal` |
| `Tab` | تدوير إكمالات أوامر الشرطة المائلة |
| `Up` / `Down` | السجل (وإخراج الموجّهات المُصطفّة) |

تُشرَح أوضاع التشغيل في [أوضاع التشغيل](../explanation/modes.md).

## أوامر الشرطة المائلة

اكتب `/` في المُحرِّر؛ ويُكمِل `Tab`. الأوامر المسجَّلة هي:

| الأمر | الغرض |
|---|---|
| `/help` | سرد الأوامر المتاحة |
| `/quit`، `/q`، `/exit` | الخروج من REPL |
| `/clear` | مسح سجل المحادثة |
| `/model` | فتح مُنتقي النماذج |
| `/mode` | تبديل وضع التشغيل (auto/planning/writing/goal) |
| `/session` | فتح مُنتقي الجلسات (استئناف) |
| `/save` | حفظ / تسمية الجلسة الحالية |
| `/history` | عرض سجل الجلسات |
| `/tokens` | استهلاك الرموز (داخل / خارج / لكل دور / لكل جلسة) |
| `/context` | حجم السياق الحالي مقابل الحد |
| `/status` | لقطة: النموذج، المزوّد، الوضع، الجلسة، مشغول، الطابور |
| `/insights` | عرض الرؤى المتعلَّمة للمشروع |
| `/rules` | عرض خلاصة قواعد المشروع |
| `/schema` | التحقق من / إصلاح `AGENTS.md` |
| `/wiki` | عمليات الويكي للبنية النشطة |
| `/daemon` | فتح لوحة التحكم بالبرنامج الخفي (المشروع → البرامج الخفية → القنوات) |

> مجموعة أوامر الشرطة المائلة هي ذاتها سواء أطلقت TUI مباشرةً أو دفعتها من
> شاشة أخرى. تكشف القنوات (مثل تيليجرام) مجموعة أوامرها الخاصة المنفصلة.

## السمات

السمات المدمجة: `everforest` (الافتراضية)، `dracula`، `gruvbox`، `tokyo-night`،
`catppuccin`. اختر واحدة بـ `Ctrl+T`، أو `veles tui --theme <name>`، أو
`[user] tui_theme` في `~/.veles/config.toml`.
