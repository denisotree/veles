# Горячие клавиши TUI и слэш-команды

> 🌐 **Языки:** [English](../../en/reference/tui.md) · [简体中文](../../zh-CN/reference/tui.md) · [繁體中文](../../zh-TW/reference/tui.md) · [日本語](../../ja/reference/tui.md) · [한국어](../../ko/reference/tui.md) · [Español](../../es/reference/tui.md) · [Français](../../fr/reference/tui.md) · [Italiano](../../it/reference/tui.md) · [Português (BR)](../../pt-BR/reference/tui.md) · [Português (PT)](../../pt-PT/reference/tui.md) · **Русский** · [العربية](../../ar/reference/tui.md) · [हिन्दी](../../hi/reference/tui.md) · [বাংলা](../../bn/reference/tui.md) · [Tiếng Việt](../../vi/reference/tui.md)

`veles tui` (или просто `veles`) открывает интерактивный REPL. Это чат со скроллбэком,
многострочным композером, строкой состояния и сворачиваемым инспектором.

## Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Ctrl+D` | Выход |
| `Ctrl+C` | Скопировать последний ответ ассистента; нажмите дважды в течение 1.5 с для выхода |
| `Ctrl+V` | Вставить из буфера обмена |
| `Ctrl+Shift+C` / `⌘C` | Скопировать текущее выделение (OSC52). В macOS Terminal.app нативное выделение мышью + ⌘C работает напрямую |
| `Ctrl+I` | Переключить инспектор (рассуждения, активность инструментов, лог токенов/ошибок) |
| `Ctrl+R` | Открыть выбор сессии (возобновить прошлую сессию) |
| `Ctrl+T` | Открыть выбор темы |
| `Shift+Tab` | Циклически сменить режим запуска: `auto → planning → writing → goal` |
| `Tab` | Циклически перебрать автодополнения слэш-команд |
| `Up` / `Down` | История (и извлечение поставленных в очередь промптов) |

Режимы запуска объяснены в [Режимах запуска](../explanation/modes.md).

## Слэш-команды

Наберите `/` в композере; `Tab` дополняет. Зарегистрированные команды:

| Команда | Назначение |
|---|---|
| `/help` | Список доступных команд |
| `/quit`, `/q`, `/exit` | Выйти из REPL |
| `/clear` | Очистить лог чата |
| `/model` | Открыть выбор модели |
| `/mode` | Переключить режим запуска (auto/planning/writing/goal) |
| `/session` | Открыть выбор сессии (возобновление) |
| `/save` | Сохранить / назвать текущую сессию |
| `/history` | Показать историю сессии |
| `/tokens` | Использование токенов (in / out / за ход / за сессию) |
| `/context` | Текущий размер контекста относительно лимита |
| `/status` | Снимок: модель, провайдер, режим, сессия, занятость, очередь |
| `/insights` | Показать выученные инсайты для проекта |
| `/rules` | Показать дайджест правил проекта |
| `/schema` | Проверить / исправить `AGENTS.md` |
| `/wiki` | Операции с вики для активного макета |
| `/daemon` | Открыть панель управления демоном (проект → демоны → каналы) |

> Набор слэш-команд один и тот же, запускаете ли вы TUI напрямую или открываете его с
> другого экрана. Каналы (например, Telegram) предоставляют свой собственный отдельный
> набор команд.

## Темы

Встроенные темы: `everforest` (по умолчанию), `dracula`, `gruvbox`, `tokyo-night`,
`catppuccin`. Выберите одну через `Ctrl+T`, `veles tui --theme <name>` или
`[user] tui_theme` в `~/.veles/config.toml`.
