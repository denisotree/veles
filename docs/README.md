# Veles documentation · Документация Veles

**Veles** is a minimalist, cleanly-decomposed CLI agent framework: a local-first
agent loop with structured project memory, a learning loop, multi-agent
orchestration, a provider-agnostic protocol, and accumulating skills/tools.

**Veles** — минималистичный, чисто декомпозированный CLI-фреймворк для агентов:
local-first агентный цикл со структурированной проектной памятью, циклом
обучения, мультиагентной оркестрацией, провайдер-независимым протоколом и
накапливаемыми навыками/инструментами.

## How the docs are organised · Как устроена документация

These docs follow the [Diátaxis](https://diataxis.fr/) framework — four kinds of
documentation for four different needs:

Документация построена по фреймворку [Diátaxis](https://diataxis.fr/) — четыре
типа документации под четыре разные задачи:

| Quadrant · Раздел | Purpose · Назначение |
|---|---|
| **Tutorials** / Туториалы | Learn by doing — one guided happy path. Учимся на практике — один проверенный путь. |
| **How-to guides** / Инструкции | Accomplish a specific task. Решить конкретную задачу. |
| **Reference** / Справочник | Look up exact commands, flags, config, env vars. Точные команды, флаги, конфиг, env. |
| **Explanation** / Концепции | Understand how and why Veles works the way it does. Понять, как и почему Veles устроен так. |
